OKZoom by OKFocus

OKZoom is a JQuery plugin that produces a portable lorgnette of variable size and shape.

Usage

Bind OKZoom to one or many image elements: 

$('img').okzoom({
  width: 200,
  height: 200,
  round: true,
  background: "#fff",
  backgroundRepeat: "repeat",
  shadow: "0 0 5px #000",
  border: "1px solid black"
});

Usage

width of the lorgnette, in pixels 150
height of the lorgnette, default is equal to width null
scaleWidth optionally resize the lorgnette image null
round circular lorgnette if true, rectangular if false true
background color for image off the edge of the lorgnette #fff
backgroundRepeat repeat the image within the lorgnette? no-repeat
border apply a border around the lorgnette 0
shadow box-shadow on the lorgnette 0 0 5px #000
