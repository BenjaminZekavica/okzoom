h1. okzoom

h2. OKZoom by OKFocus

p. OKZoom is a jQuery plugin that produces a portable lorgnette of variable size and shape. All other jQuery 'zoom' plugins we have encountered implement a square magnifying area. Ours is a circle. You want a circle. See a demo "here":http://okfoc.us/okzoom.

h2. Usage

p. You can bind OKZoom to one or many image elements, producing a "zoom" effect like this:

!https://github.com/okfocus/okzoom/raw/master/demos/img/okshadow-example.png!

p. With code like this:

<pre class='js'>
<code>
$('img').okzoom({
  width: 200,
  height: 200,
  round: true,
  background: "#fff",
  backgroundRepeat: "repeat",
  shadow: "0 0 5px #000",
  border: "1px solid black"
});
</code>
</pre>

h3. Options

|Usage|||
|width|(in pixels}|150|
|height|(in pixels}|150|
|scaleWidth|optionally resize the lorgnette image|null|
|round|round if true, square if false|true|
|background|color for image off the edge of the lorgnette|#fff|
|backgroundRepeat|repeat the image within the lorgnette|no-repeat|
|border|border around the lorgnette|0|
|shadow|box-shadow on the lorgnette|0 0 5px #000|
